Project: Pick-Up

Technologies: React Native, Node.js, JS, TS, Figma, AWS RDS, MySQL, Caddy Server, AWS EC2

Description: Pick-Up is a mobile application built to bring basketball players together onto one platform to compete and find their rank amongst their peers. Using the ELO rating system and player honesty level we hope to create leaderboards that accurately quantify basketball players skill level relative to their competitors. Using each players Honesty Level in conjuction with cross-verification of scores uploaded by both players after each match we are able to punish players who frequently input false info and reward those who dont.

Features: Account creation, one time passcode(OTP) verification, inter-user interaction, personalized leaderboards, global leaderboards, match history, match score cross-verifiction, honesty incentives/disincentives
